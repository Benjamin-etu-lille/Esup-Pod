# Esup-Pod v4 Stand-alone installation

## Environment

### User creation in Pod

```sh
user@pod:~$  sudo adduser pod
user@pod:~$  adduser pod sudo
user@pod:~$  su pod
user@pod:~$  cd /home/pod
```

### Packages installation

```sh
pod@pod:~$ sudo apt-get install git curl vim
```
